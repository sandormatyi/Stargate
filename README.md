# Stargate
Szoftver projekt laboratórium tárgy házi feladata

# Instructions
Compile and run the game with the included .bat files (JRE 1.6+ required).

## Player 1 controls:
* Movement: W, A, S, D
* Portal gun: F, G
* Pick up box: Space

## Player 2 controls:
* Movement: Arrow keys
* Portal gun: Numpad 2, 3
* Pick up box: Numpad 1
